## ⚙️ How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/IndeewaraC/Personal-Portfolio---Indeewara-Gunathilaka.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Personal-Portfolio---Indeewara-Gunathilaka
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
